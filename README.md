# Custom CNN Face Detector

A face detection system built from scratch using PyTorch — no Haar cascades, no pretrained weights, no detection libraries.

## How it works

1. **Patch classifier** — a small CNN is trained to answer one binary question: does this 64×64 image patch contain a face?
2. **Sliding window** — at inference time the classifier is run over every position in a frame across multiple scales.
3. **NMS** — overlapping high-confidence boxes are collapsed into single detections.

## Project structure

```
facedetec/
├── dataset.py        # LFW download, negative mining, FaceDataset, DataLoaders
├── model.py          # FaceDetectorCNN (3 conv blocks + classifier head)
├── train.py          # Training loop, validation, checkpointing, loss curves
├── detect.py         # Sliding window + NMS — Detector class
├── main.py           # Webcam feed entry point
├── config.yaml       # All hyperparameters and paths
├── requirements.txt
└── data/
    ├── positives/    # 64×64 face crops (generated by dataset.py)
    ├── negatives/    # 64×64 background patches (generated by dataset.py)
    └── backgrounds/  # Raw background images (user-supplied)
```

## Quickstart

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Prepare the dataset  (downloads LFW, mines negatives)
python dataset.py

# 3. Train the model
python train.py

# 4. Run live detection
python main.py
```

## Configuration

All hyperparameters live in `config.yaml`. Key values:

| Key | Default | Description |
|-----|---------|-------------|
| `training.epochs` | 30 | Number of training epochs |
| `training.batch_size` | 64 | Batch size |
| `training.learning_rate` | 0.001 | Initial Adam LR |
| `detection.detection_threshold` | 0.85 | Min confidence to accept a box |
| `detection.nms_iou_threshold` | 0.3 | IoU cutoff for NMS |
| `detection.scales` | [1.0, 0.75, 0.5, 0.25] | Image pyramid scales |
| `detection.step_size` | 16 | Sliding window stride (px) |

## Model architecture

```
Input (3 × 64 × 64)
  └─ Conv(32) → BN → ReLU → MaxPool   →  32 × 31 × 31
  └─ Conv(64) → BN → ReLU → MaxPool   →  64 × 15 × 15
  └─ Conv(128)→ BN → ReLU → MaxPool   → 128 ×  6 ×  6
  └─ Flatten → Linear(4608→512) → ReLU → Dropout(0.5)
  └─ Linear(512→1) → Sigmoid
```

## Dataset

**Positive samples** — LFW (Labeled Faces in the Wild), ~13 000 face crops downloaded automatically via `sklearn.datasets.fetch_lfw_people`.

**Negative samples** — random 64×64 patches mined from background images you place in `data/backgrounds/`. Any images without faces work well (COCO backgrounds, indoor/outdoor photos, etc.).
